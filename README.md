# COMP7710 Practice Exam Repository

## Getting Started

This repository has been configured and tested using IntelliJ IDEA. The `.idea/` directory and `practice-exam.iml` file are included to help you get started quickly by simply cloning the repository and opening it in IntelliJ.

The included IntelliJ configuration is provided as a convenience only and is not required for completing the exercises.

For most students, the following should be sufficient:

1. Clone the repository.
2. Open the project folder in IntelliJ IDEA.
3. Allow IntelliJ to load the project configuration.
4. Run the test cases.

## Troubleshooting

IntelliJ project settings are sometimes machine-specific. If the project does not open correctly, or if the test cases fail to run due to configuration issues, you may safely delete:

* `.idea/`
* `practice-exam.iml`

and configure the project again from scratch in your local IntelliJ installation.

## Required Libraries

To run the provided JUnit test cases, your project must include the following libraries:

* JUnit
* Hamcrest Core

If these libraries are missing, IntelliJ will typically display compilation errors in the test files. You can add them through:

`File → Project Structure → Libraries`

or by using IntelliJ's automatic dependency resolution when opening the project.

## Extra TODO

As part of your exam preparation, try to read and understand the provided test cases in the `*Test.java` files.

The test cases are designed to clarify the expected behaviour of each method and often provide useful examples of valid inputs, outputs, and edge cases. Developing the habit of reading tests can help you better understand specifications and improve your problem-solving skills during the exam.
